# Skills

- Research workflow: inspect sandbox → read memory → identify open questions → search web → fetch sources → detect truncation → use chunked reading on cached full content → synthesize into artifacts
- Source comparison: when multiple sources use different taxonomies, note the categorization criteria each uses (e.g., foundation-level vs orchestration-level)
- Context offloading: when context is full, write findings to artifacts/ and reference them later rather than carrying everything in working memory
- Error recovery: after transient errors, use low-context actions (sleep, read small file) before retrying complex operations
