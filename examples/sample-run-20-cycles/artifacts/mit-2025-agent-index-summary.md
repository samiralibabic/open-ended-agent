# MIT 2025 AI Agent Index - Summary

**Source**: https://aiagentindex.mit.edu/

## Key Findings

### Deployment Pace

- 24/30 agents launched or received major agentic updates in 2024-2025
- Research attention on "AI agent" in 2025 exceeded all prior years combined

### Autonomy Levels (L1-L5)

- **Chat agents**: L1-L3 (turn-based interaction)
- **Browser agents**: L4-L5 (limited mid-execution intervention)
- **Enterprise agents**: L1-2 in design, L3-5 when deployed (event-triggered, no human in loop)

### Safety Transparency Gap

- 4/13 frontier agents disclose agentic safety evaluations
- 23/30 agents have no third-party testing
- Only 4 agents provide agent-specific system cards
- SOC 2 / ISO 27001 more common than agent-specific frameworks

### Foundation Model Concentration

- Almost all agents depend on GPT, Claude, or Gemini families
- Only frontier labs and Chinese developers run own models

### Tool Integration

- 20/30 agents support MCP (Model Context Protocol)
- Enterprise agents leading at 12/13

### Open Source

- 23/30 agents fully closed source at product level

### Evaluation

- Multi-layered ecosystem makes evaluation difficult
- Model-level evaluation is insufficient for agentic systems
- Agentic evaluations depend on downstream context and tools
